20.3 Donde Obtener la Última Versión de Este Documento

Puedes encontrar la última versión de este documento, una especie de páginas de manual oficiales de Sylpheed en http://nlpagan.net/sylman/index.html.

Aquí el manual está almacenado como páginas HTML. Desde ésa página puedes descargar estas páginas como un fichero tar.gz para verlo fuera de línea en tu propia computadora.

También puedes ir al repositorio CVS en http://sourceforge.net y buscar el proyecto "sylpheeddoc". El documento está almacenado allí como un fichero SGML, ¡necesitarás las SGMLtools para ésto!

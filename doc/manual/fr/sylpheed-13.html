13. Support des types MIME

13.1 Comment Sylpheed gère les types MIME

(Merci à Adriaan Penning)

Deux fichiers, mime.types et mailcap sont utilisés pour déterminer le type MIME d'une pièce jointe ainsi que l'application associée à ce type. Le fichier mime.types est utilisé pour déterminer le type MIME d'une pièce jointe que vous ajoutez lors de la composition d'un message. Le fichier mime.types contient des lignes telles que: 

application/pdf                    pdf

Sylpheed recherche ce fichier dans le répertoire défini par la variable SYSCONFDIR (définie à la compilation), dans /etc et enfin dans $HOME/.sylpheed/. Si ce fichier n'existe pas ou n'est pas dans le bon format, toutes les pièces jointes auront le type application/octet-stream et le destinataire du message ne saura pas quelle application utiliser pour voir ces pièces jointes.

Pour définir l'application à utiliser pour visualiser une pièce jointe, il suffit d'éditer le fichier mailcap (/etc/mailcap ou $HOME/.mailcap).

Par example, vous devez ajouter les lignes suivantes dans le fichier $HOME/.mailcap pour utiliser xpdf pour visualiser les fichiers au format PDF : 


application/pdf ; xpdf %s

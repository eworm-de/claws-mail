19.2 Liste de diffusion

Si vous voulez garder Sylpheed à jour et que vous voulez discuter du programme avec d'autres utilisateurs, vous pouvez rejoindre la liste de diffusion de Sylpheed.

Envoyez un message à sylpheed-ctl@good-day.net contenant dans le corps du message le texte suivant :

subscribe Votre nom

Le serveur de la liste de diffusion vous retournera le courrier une fois pour le premier enregistrement de la requête vous demandant une confirmation disant à peu-près : « Puis-je vous mettre sur la liste de diffusion ? ». Il inclut le texte suivant (le nombre et le nom sont un exemple) :

confirm 84682771 Anne Shirley

C'est une précaution pour éviter que vous soyez inscrit à votre insu.

Lorsque vous recevez ce courrier de confirmation, envoyez un courrier qui inclut la ligne de précédente :

confirm MotDePasse(le nombre) Votre nom

à la même adresse que précédemment (sylpheed-ctl@good-day.net). Une fois cette confirmation envoyée, vous serez inscrit dans la liste de diffusion.

L'adresse pour poster dans la liste est différente de celle pour l'enregistrement et l'envoi de commandes. En effet, il s'agit de sylpheed@good-day.net.

Notez que si vous perdez le courrier contenant la demande de confirmation, envoyez simplement de nouveau une demande d'inscription :

subscribe Votre nom

à sylpheed-ctl@good-day.net.

Comment annuler son inscription ?

Envoyez simplement un courrier à l'adresse sylpheed-ctl@good-day.net contenant uniquement la ligne suivante :

unsubscribe

19.4 Comment indiquer à un autre programme d'utiliser Sylpheed comme programme de gestion de courriers ?

Si vous souhaitez utiliser Sylpheed depuis Netscape ou Opera, ouvrez la fenêtre Preferences du navigateur, puis choisissez Applications. Dans la partie Email client entrez sylpheed --compose.
